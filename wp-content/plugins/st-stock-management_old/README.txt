== Description ==

ST Stock Management is a comprehensive WordPress plugin for managing your company's inventory, assets, and employee assignments. Features include:

* Item Management with asset types and detailed item tracking
* Employee Management with position and status tracking
* Asset Assignment system for employees
* Repair Management (coming soon)
* Bootstrap UI with DataTables integration
* AJAX operations with SweetAlert notifications

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/st-stock-management` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. The plugin will create the necessary database tables automatically
4. Access the management system from the admin menu 'Stock Management'

== Usage ==

Admin Features:
- Navigate to Stock Management in WordPress admin
- Manage Asset Types, Items, and Employees
- View all data with search and pagination

Frontend Features:
- Use shortcode [stock_management] on any page/post
- Access item and employee management on frontend
